# My Nvim Config

Just my Neovim config if someone needs it or wants to make fun of it. I use vim-plug on an Arch Linux machine.

Also I use the following Coc Plugins:
- coc-pairs
- coc-pyright
- coc-clangd
- coc-snippets

(The theme that I personally use is in the config file)
